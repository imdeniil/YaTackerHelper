# Alembic Migrations

Папка с миграциями базы данных для проекта YaTackerHelper.

## Быстрый старт

```bash
# Применить все миграции
uv run alembic upgrade head

# Откатить последнюю миграцию
uv run alembic downgrade -1

# Создать новую миграцию
uv run alembic revision -m "description"
```

## Структура

- `versions/` - Файлы миграций (коммитятся в git)
- `env.py` - Конфигурация окружения, загружает DATABASE_URL из .env
- `script.py.mako` - Шаблон для новых миграций

## Подробная документация

См. `docs/MIGRATIONS.md` для полной документации по работе с миграциями.

## Важно

- Всегда проверяйте сгенерированные миграции перед применением
- Делайте backup БД перед применением миграций на продакшене
- Файлы миграций должны быть в git
